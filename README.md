# PararealImageViewer

VketReal/Parareal活動向け全画面画像表示アプリ

## 用途

PCに指定したフォルダ内の画像を全画面で表示します。

Parareal活動に最適化して操作UIを上部にまとめています。

## 注意事項

- 開発機の1920x1080の解像度で動作確認しています。
- 対応画像ファイルは jpg/jpeg/png です。
- タッチ操作でアプリの切り替えは別アプリ「PararealControlPanel」を併用する前提です。

## 開発環境

Microsoft Visual Studio Community 2022 (64 ビット) - Current

C# / .NET Framework 4.8.1

## 誰が使うねん

VketRealにVR対応タブレットを持ってきて活動する人
